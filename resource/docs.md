# 安装
```bash
// install node.js 
node -v //查看node版本 
npm -v //查看npm版本 
npm i docsify-cli -g 
```
# 内容编排
docsify init docs //初始化 \
README为首页展示文件，index.html中编写配置项(需熟悉html、css、javascript) 

# 部署
docsify serve docs -p 4000 //打开服务器 

# 配置
支持latex公式：简单来说，用js脚本解析、展示
